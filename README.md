# Game

The game is a rouge like game.

## Storie

We are on a mission for the great kingdom of Aarhus University

## Design is 2D

## Roles

* Four different roles

  * Mage
  * Range
  * Summoner

* Block
* Dashing
* Steah
* Health System
* Mana or Energi
